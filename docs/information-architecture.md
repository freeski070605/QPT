# Information Architecture

## Top-Level Navigation
1. Home
2. Gallery
3. Drops
4. Artist
5. Collectors
6. Admin

## Primary Routes
1. `/` Home
2. `/gallery` Collection gallery
3. `/artwork/[slug]` Artwork detail
4. `/drop` Drop countdown and waitlist
5. `/artist` Bio, philosophy, press
6. `/account` Collector account
7. `/admin` Admin shell

## Conversion Path
1. Instagram or social referral
2. Home hero CTA
3. Gallery filters
4. Artwork detail page
5. Checkout method selection
6. Order confirmation
7. Collector account follow-up

## Admin Information Areas
1. Inventory management
2. Order verification
3. Content management
4. Drop scheduling
5. Analytics dashboard

## Content Domains
1. Artwork catalog
2. Artist story and process
3. Drops and waitlist
4. Social proof and testimonials
5. Collector resources
